## Task Tracker 
- A simple task management application built with Vanilla JavaScript that allows users to add , complete , delete and persist tasks using the browsers Local Storage.

 #Features 
 - add task
 - remove task
 - complete task
 - clear all tasks
 - clear completed tasks
 - tasks persist using LocalStorage
 - responsive layout
   
#Technologies used
- Html
- JavaScript
- Css
- Local Storage API

   ## how to use
   - open index.html on your browser
   - enter a task in the input field
   - confirm task by pressing 'Add Task' or Enter on your keyboard
   - use checkbox to mark tasks as complete
   - delete or clear tasks as needed.
